# PWA Icons Required

The manifest.json requires the following icon files to be added to this directory:

## Required Files:

1. **icon-192x192.png** (192x192 pixels)
   - Purpose: Any (standard icon)
   - Used for: Home screen icon on mobile devices

2. **icon-512x512.png** (512x512 pixels)
   - Purpose: Any (standard icon)
   - Used for: Splash screens and larger displays

3. **icon-192x192-maskable.png** (192x192 pixels)
   - Purpose: Maskable (adaptive icon)
   - Safe zone: 80% of the canvas (38px margin on all sides)
   - Used for: Android adaptive icons

4. **icon-512x512-maskable.png** (512x512 pixels)
   - Purpose: Maskable (adaptive icon)
   - Safe zone: 80% of the canvas (102px margin on all sides)
   - Used for: Android adaptive icons on larger displays

## Design Guidelines:

- **Brand Colors**: Use #c11820 (primary red) and white
- **Logo**: Include SITIMM logo/text centered
- **Maskable Icons**: Keep important content within the safe zone (80% center area)
- **Format**: PNG with transparent or white background
- **Quality**: High resolution, optimized for web

## Tools to Create Icons:

- [Maskable.app](https://maskable.app/) - Test maskable icons
- [PWA Asset Generator](https://github.com/onderceylan/pwa-asset-generator)
- Figma/Photoshop/Illustrator for design
